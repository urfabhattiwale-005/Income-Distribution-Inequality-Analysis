# Income-Distribution-Inequality-Analysis
This project analyzes income distribution using IRS SOI data and applies statistical modeling techniques to measure inequality. It uses the Lorenz Curve and Gini Coefficient to evaluate income disparity, along with visualizations and income share analysis to understand how income is distributed across population groups.

Income Distribution & Inequality Analysis
Overview:
This project analyzes income distribution using IRS SOI data and measures inequality using statistical modeling techniques.

Objective:
To model income distribution and measure inequality using Lorenz Curve and Gini Coefficient.

Dataset:
IRS SOI Tax Stats Dataset

Tools Used:
Python
pandas
numpy
matplotlib

Key Analysis:
Data cleaning and preprocessing
Income distribution modeling
Lorenz Curve visualization
Gini Coefficient calculation
Income share analysis

Results:
Income distribution is highly skewed
Higher income groups dominate total income
Gini coefficient indicates inequality

Conclusion:
The project successfully demonstrates income inequality using statistical methods and real-world data.
